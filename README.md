# ThumbnailView

ThumbnailView 스위프트 패키지 입니다.

## Overview

### Requirements

- macOS Ventura 13.0
- Xcode 14.0.1
- iOS 13.0

### Installation

#### Swift Package Manager

```swift
.package(url: "https://github.com/jaemyeong/ThumbnailView.git", .upToNextMajor(from: "0.1.2"))
```

### Usage

```swift
import ThumbnailView

let thumbnailView = ThumbnailView()
```
